Sanrakshak

Sanrakshak aims to design and develop a real-time emergency response system that enhances personal safety through instant alert triggering, live location tracking, and intelligent monitoring using modern web technologies.

Problem Statement

Personal safety remains a significant concern in urban environments, particularly for women, elderly individuals, and students. Existing safety applications provide basic SOS alerts but lack real-time responsiveness, centralized monitoring, and scalable web accessibility. There is a critical need for a unified safety platform that ensures instant alert activation, live location tracking, and quick emergency communication. Sanrakshak addresses this challenge through an intelligent web-based emergency response system.

Literature Review

We studied research on mobile emergency response systems, HCI principles in distress scenarios, and GPS-based alert frameworks. Our literature review focused on usability constraints and architectural limitations in current safety applications.

Research Gap & Innovation

Most existing systems stop at sending an SOS notification. Our approach extends beyond alert transmission by introducing escalation logic, structured monitoring, and a dashboard-based framework that improves visibility and response coordination. The innovation lies in system design and usability rather than just the alert feature.

System Methodology

Input Data

GPS location data, emergency trigger inputs, and real-time alert status updates.

Architecture

React.js frontend, Node.js backend, REST API integration, and real-time event handling for alerts.
